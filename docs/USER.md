# 使用手冊（User）

給實際使用 Bot／Mini App 的人。部署與變數請看 [DEPLOY.md](DEPLOY.md)。

## 第一次使用

1. 管理員完成 Deploy，並設好白名單（你的 Telegram User ID）
2. 在 Telegram 打開 Bot
3. 確認輸入框左側有 **開啟 App**
4. 下方應有鍵盤：**狀態**／**列表**／**說明**
5. 點 **開啟 App** 進入 Mini App

若沒有左側按鈕或鍵盤：傳任意訊息給 Bot（或請管理員重跑 Deploy）。

---

## Mini App

### 看下載
- 進入後自動列出種子，約每 4 秒更新
- 可按重整；可依加入時間、名稱、進度、速度、大小、ETA 排序

### 加下載
1. 貼上 magnet 或 torrent 網址
2. （可選）選分類
3. 送出  

也可用「貼上」從剪貼簿帶入。

> Mini App **不能**選本機 `.torrent` 檔。請改傳檔案給 Bot。

### 單筆操作
每一列可：暫停／繼續、改分類、移除（只刪任務）、刪除（連檔案一起刪，會再確認）。

### 批次
1. 點「批次」進入選取模式  
2. 勾選多筆  
3. 暫停／繼續／移除／刪檔

## Bot

### 鍵盤按鈕

| 按鈕 | 作用 |
|------|------|
| 狀態 | 總數、下載中／做種／暫停、總上下載速度 |
| 列表 | 最多 15 筆進行中種子 |
| 說明 | 使用說明 |

仍可輸入 `/start`、`/status`、`/list`、`/help`（效果相同）。

### 加種方式
- 直接傳 **magnet** 或 **torrent URL** 文字
- 傳 **`.torrent` 檔案**

成功會回「已加入…」。

### 左側「開啟 App」
開啟同一個 Mini App 網頁（Workers URL）。

---

## 通知

| 通知 | 何時 |
|------|------|
| ⬇️ 下載開始 | 新種子加入後約 0–2 分鐘內（Cloudflare Cron） |
| ✅ 下載完成 | 完成後約 0–2 分鐘內 |

注意：

- 透過 Bot 加種時，會先有「已加入」，之後可能再收到「下載開始」
- 超過約 15 分鐘才被系統掃到的事件可能不再通知
- 通知只會寄給白名單內的帳號（通常是你自己）

## 常見狀況

**顯示未授權**  
你的 Telegram ID 不在白名單，請管理員把 ID 加入 `ALLOWED_TELEGRAM_USER_IDS` 並重新 Deploy。

**Mini App 打不開／白屏**  
確認用 Telegram 內「開啟 App」進入，不要只用瀏覽器開 Workers 網址（缺少 Telegram 驗證資料）。

**點了沒反應／列表空的**  
qBittorrent 可能連不上，或帳密／URL 設定錯誤；請管理員查 Worker／Actions 日誌。

**沒有開始／完成通知**  
確認 Cloudflare Worker 已啟用 Cron（`*/2 * * * *`），且 `CRON_SECRET`、白名單正確；可手動 `POST /api/cron/completions` 測試。

**誤刪檔案**  
「刪除」會連下載檔一起刪；只要從列表移除請用「移除」。
